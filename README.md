# Blink-it--analysis-
Interactive Power BI dashboard analyzing $1.2M in Blinkit outlet sales across item types, outlet tiers, and establishment years.
This project analyzes sales performance data from Blinkit — India's leading quick-commerce platform. The dashboard provides a comprehensive view of outlet-level sales across different city tiers, store types, item categories, and fat content segments. Built entirely in Power BI with DAX measures and Power Query transformations, it enables business stakeholders to explore KPIs dynamically using slicers for outlet location, size, and item type.
